# city-explorer-api

**Author**: Drew Stroede
**Version**: 1.1.0

## Overview
This application is a website that can give you information on any city that you enter. the application can provide information such as: map, weather, lattitude/longitude and more!

## Getting Started
In order to start building an app like this you must create a new repository and add the necessary react/bootstrap/axios dependencies. From here you will also need to add an api key from location IQ.

## Architecture
This app features javascript/react/bootstrap/axios/locationIQ/vite. This app also uses servers albeit only statically for now.
